Windows Driver Setup

Windows needs a WinUSB driver installed before the web flasher can communicate with the device. Follow these steps:

  1. If the Ingenic vendor USB driver is installed, remove it first via Device Manager.
  2. Connect the device in USB boot mode.
  3. Download and run Zadig .
  4. In Zadig, select Ingenic USB Boot Device from the dropdown.
  5. Set the target driver to WinUSB and click Install (or Replace Driver).
  6. Return to this page and click Connect Device.
Important: The Ingenic vendor driver (libusb0.sys) is not compatible and must be removed before installing WinUSB via Zadig.
